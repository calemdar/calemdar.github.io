Tune Mountain


Tune Mountain is an audiovisual snowboarding game. This game is my senior thesis for university, and is still under development. The main premise of the game is that you get to choose which song you want to listen to, through Spotify, and the game creates a procedural mountain for you to snowboard down on.

I am the lead designer, engine programmer, and the project manager of this project. I came up with the idea of gamifying an audio visualizer through procedural terrain generation to create a virtual environment that enhances the listening experience. This project was created with the help of two other developers, and two artists.
