HCI


Human-Computer Interaction is a field of computer science that focuses on how us humans interact with a piece of software psychologically, socially, viscerally... It studies the humanistic factors related to a communication between a person and a computer. In this class we had to rapidly produce functional prototypes to be able to user test them with our classmates.

There was a variety of different projects, utilizing different platforms, languages, and libraries. After every project submission, the groups would completely change for the next project. This forced us to work with many different group dynamics, creating things we had no idea how to create. I put this class as a project here because I think assembling a group of people who have never met, and being able to produce testable prototypes withing two weeks is a useful skill that most schools don't care about. I learned a lot about how to learn to use a software or framework that I never used before. I also learned a lot about how to get a team to start being productive right away. If you want to see more about the projects, you can check out my design manifesto article.
