Stick It!

This game uses a JS based web engine called Perlenspiel. The engine utilizes a game grid of size up to 32x32 squares. Each square is a bead object that can do many things. Stick It! utilizes a hub world to diminish linearity, provide an environment to test game mechanics, and encourage replayability. The game was designed around two core mechanics: Leaving a mark and sticking. These two mechanics gave us ideas of different ways to get from point A to point B; in this case the starting position and the end goal.

The game has ten different levels, and a hub world to navigate to the levels. The source code of the game can be found in the Github link below. All of the code is written by me in two weeks. We user tested the game with thirty different players, gathered feedback, and polished. Older versions of the game can be found on the same website.
